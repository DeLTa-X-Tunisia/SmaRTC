# SmaRTC Documentation

Welcome to the official documentation for SmaRTC, a complete, self-hosted platform for real-time communication. This documentation provides a comprehensive guide to the architecture, deployment, and development of the SmaRTC platform.

## Table of Contents

1.  **[Architectural Overview](./README.md)**
    -   A high-level look at the microservices-based architecture and the role of each component.

2.  **[Docker Startup Guide](./docker-startup.md)**
    -   Detailed instructions for setting up, running, and managing the application stack using Docker Compose.

3.  **[Development Guide](./development-guide.md)**
    -   Essential information for developers, including project structure, running tests, managing database migrations, and coding conventions.

4.  **[Troubleshooting Archive](./troubleshooting.md)**
    -   A log of key issues encountered during development and their resolutions, providing valuable insights for future debugging.

## API Documentation

The SmaRTC API is documented using Swagger/OpenAPI. When the application is running, you can access the interactive API documentation by navigating to the `/swagger` endpoint of the API service.
